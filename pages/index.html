TokenEx 3DS Demo App

Overview

This is a demo of the TokenEx 3DS Product. Select a device fingerprinting option below to get started.

TokenEx device fingerprinting is handled through the TokenEx Iframe and adds two values to the configuration: use3DS:true and threeDSMethodNotificationUrl for receiving the notification when device fingerprinting is completed. A TokenEx Iframe notice is also raised when device fingerprinting is completed. Either the notification or the notice can be used to trigger an Authentication API call.

Manual device fingerprinting happens outside the TokenEx Iframe (simply used for tokenizing the PAN in this scenario) and is managed by the merchant. Both demos can be used to assist with developing your application's authentication flow.
